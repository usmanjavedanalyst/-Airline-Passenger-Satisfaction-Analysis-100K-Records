##  Overview
This project focuses on analyzing **airline passenger satisfaction** using an interactive **Excel dashboard**.  
With a dataset of **100,000+ passenger records**, the goal was to turn raw data into **clear, business-ready insights**.

---

##  What This Project Covers
- Passenger **satisfaction vs dissatisfaction**
- Impact of **arrival & departure delays**
- Comparison across **Business, Economy & Eco Plus**
- **Loyal vs disloyal** customer behavior
- Key service ratings (comfort, cleanliness, inflight service)

---

##  Key Insight
 Flights delayed **more than 15 minutes** show a noticeable drop in passenger satisfaction, making delays one of the strongest drivers of negative feedback.
